A full-stack web app that allows you to add, edit, view pets and vets. You can also add schedules to you pet's medication.

## Tech Stack
- **Frontend:** Next.js, React, Shadcn, Tailwind CSS, Zod  + TanStack Query for fetching & caching data

## Features
- Add, view pets
- Add, view vets
- Validator: Zod
- Track schedule and medications
- UI built with Tailwind CSS
- Uses MongoDB for data storage and Next.js for frontend/backend.
- Utilized shadcn for UI and Tanstack Query for hooks
